# Avalonia Tools

Avalonia is a free, open-source UI framework. You can build and ship cross-platform .NET applications at no cost. The framework is MIT licensed and maintained by a growing team.

Development tooling covers the tasks around the framework itself: diagnosing layout issues, packaging your app for multiple operating systems, and previewing XAML as you write it.

## Accelerate[​](#accelerate "Direct link to Accelerate")

Avalonia Accelerate is a suite of professional tools built specifically for Avalonia development.

[![](/icons/feature-accelerate-devtools-icon.png)](/tools/developer-tools/installation.md)

[Dev Tools](/tools/developer-tools/installation.md)

[Inspect and diagnose your Avalonia apps visually. Edit properties in real time, profile performance, and debug layouts without guesswork.](/tools/developer-tools/installation.md)

[![](/icons/feature-accelerate-parcel-icon.png)](/tools/parcel/setup.md)

[Parcel](/tools/parcel/setup.md)

[Package your apps for Windows, macOS, and Linux in a single tool. Code signing, notarisation, and installers handled for you.](/tools/parcel/setup.md)

[![](/icons/feature-accelerate-vs-ext-icon.png)](/tools/visual-studio-extension.md)

[Avalonia for Visual Studio](/tools/visual-studio-extension.md)

[A purpose-built Visual Studio extension with XAML previewing, code completion, and a drag-and-drop designer.](/tools/visual-studio-extension.md)

Accelerate also includes UI controls such as [TreeDataGrid](/controls/data-display/structured-data/treedatagrid.md), [NativeWebView](/controls/web/nativewebview.md), [VirtualKeyboard](/controls/input/text-input/virtualkeyboard.md), and more. These components cover use cases from displaying hierarchical data to embedding native web content without bundling Chromium.

## Who gets access[​](#who-gets-access "Direct link to Who gets access")

If you work at a company with fewer than five developers and less than $1M USD in annual revenue, the [Community Edition](/tools/community-edition.md) gives you free access to all Accelerate tools and components. No trial period, no feature gates.

For larger teams and organisations, paid subscriptions are available in Business and Enterprise tiers. Business includes everything most teams need. Enterprise adds specialised components like VirtualKeyboard for kiosk and embedded scenarios.

Subscriptions fund continued development of the open-source framework.
