# Provider Namespace

Namespace `Avalonia.Automation.Provider`

## Types[​](#types "Direct link to Types")

* [IEmbeddedRootProvider](/api/avalonia/automation/provider/iembeddedrootprovider.md) - Exposes methods and properties to support UI Automation client access to the root of an automation tree hosted by another UI framework.
* [IExpandCollapseProvider](/api/avalonia/automation/provider/iexpandcollapseprovider.md) - Exposes methods and properties to support UI Automation client access to controls that visually expand to display content and collapse to hide content.
* [IInvokeProvider](/api/avalonia/automation/provider/iinvokeprovider.md) - Exposes methods and properties to support UI Automation client access to controls that initiate or perform a single, unambiguous action and do not maintain state when activated.
* [IRangeValueProvider](/api/avalonia/automation/provider/irangevalueprovider.md) - Exposes methods and properties to support access by a UI Automation client to controls that can be set to a value within a range.
* [IRootProvider](/api/avalonia/automation/provider/irootprovider.md) - Exposes methods and properties to support UI Automation client access to the root of an automation tree.
* [IScrollProvider](/api/avalonia/automation/provider/iscrollprovider.md) - Exposes methods and properties to support access by a UI Automation client to a control that acts as a scrollable container for a collection of child objects.
* [ISelectionItemProvider](/api/avalonia/automation/provider/iselectionitemprovider.md) - Exposes methods and properties to support access by a UI Automation client to individual, selectable child controls of containers that implement [Avalonia.Automation.Provider.ISelectionProvider](xref:Avalonia.Automation.Provider.ISelectionProvider).
* [ISelectionProvider](/api/avalonia/automation/provider/iselectionprovider.md) - Exposes methods and properties to support access by a UI Automation client to controls that act as containers for a collection of individual, selectable child items.
* [IToggleProvider](/api/avalonia/automation/provider/itoggleprovider.md) - Exposes methods and properties to support UI Automation client access to controls that can cycle through a set of states and maintain a particular state.
* [IValueProvider](/api/avalonia/automation/provider/ivalueprovider.md) - Exposes methods and properties to support access by a UI Automation client to controls that have an intrinsic value that does not span a range and that can be represented as a string.
* [ScrollAmount](/api/avalonia/automation/provider/scrollamount.md)
* [ToggleState](/api/avalonia/automation/provider/togglestate.md) - Contains values that specify the toggle state of a UI Automation element.
