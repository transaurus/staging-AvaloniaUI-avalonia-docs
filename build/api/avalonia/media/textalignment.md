#### Remarks[​](#remarks "Direct link to Remarks")

When the [TextAlignment](xref:Avalonia.Media.TextAlignment) property is set to DetectFromContent, alignment is inferred from the text content of the control. For example, English text is left aligned, and Arabic text is right aligned.
