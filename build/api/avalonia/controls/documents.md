# Documents Namespace

Namespace `Avalonia.Controls.Documents`

## Types[​](#types "Direct link to Types")

* [Bold](/api/avalonia/controls/documents/bold.md) - Bold element - markup helper for indicating bolded content. Equivalent to a Span with [FontWeight](xref:Avalonia.Media.FontWeight) property set to FontWeights.Bold. Can contain other inline elements.
* [Inline](/api/avalonia/controls/documents/inline.md) - Inline element.
* [InlineCollection](/api/avalonia/controls/documents/inlinecollection.md) - A collection of [Avalonia.Controls.Documents.Inline](xref:Avalonia.Controls.Documents.Inline)s.
* [InlineUIContainer](/api/avalonia/controls/documents/inlineuicontainer.md) - [InlineUIContainer](xref:Avalonia.Controls.Documents.InlineUIContainer) - a wrapper for embedded UIElements in text flow content inline collections
* [Italic](/api/avalonia/controls/documents/italic.md) - Italic element - markup helper for indicating italicized content. Equivalent to a Span with [FontStyle](xref:Avalonia.Media.FontStyle) property set to FontStyles.Italic. Can contain other inline elements.
* [LineBreak](/api/avalonia/controls/documents/linebreak.md) - LineBreak element that forces a line breaking.
* [Run](/api/avalonia/controls/documents/run.md) - A terminal element in text flow hierarchy - contains a uniformatted run of unicode characters
* [Span](/api/avalonia/controls/documents/span.md) - Span element used for grouping other Inline elements.
* [TextElement](/api/avalonia/controls/documents/textelement.md) - [TextElement](xref:Avalonia.Controls.Documents.TextElement) is an base class for content in text based controls. TextElements span other content, applying property values or providing structural information.
* [Underline](/api/avalonia/controls/documents/underline.md) - Underline element - markup helper for indicating superscript content. Equivalent to a Span with [TextDecorations](xref:Avalonia.Media.TextDecorations) property set to TextDecorations.Underlined. Can contain other inline elements.
