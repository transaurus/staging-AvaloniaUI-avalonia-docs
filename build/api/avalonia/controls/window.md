#### Remarks[​](#remarks-3 "Direct link to Remarks")

If [Avalonia.Controls.Window.SizeToContent](xref:Avalonia.Controls.Window.SizeToContent) has a value other than [Avalonia.Controls.SizeToContent.Manual](xref:Avalonia.Controls.SizeToContent.Manual), [Avalonia.Controls.Window.SizeToContent](xref:Avalonia.Controls.Window.SizeToContent) is automatically set to [Avalonia.Controls.SizeToContent.Manual](xref:Avalonia.Controls.SizeToContent.Manual) if a user resizes the window by using the resize grip or dragging the border.

NOTE: Because of a limitation of X11, [Avalonia.Controls.Window.SizeToContent](xref:Avalonia.Controls.Window.SizeToContent) will be reset on X11 to [Avalonia.Controls.SizeToContent.Manual](xref:Avalonia.Controls.SizeToContent.Manual) on any resize - including the resize that happens when the window is first shown. This is because X11 resize notifications are asynchronous and there is no way to know whether a resize came from the user or the layout system. To avoid this, consider setting [Avalonia.Controls.Window.CanResize](xref:Avalonia.Controls.Window.CanResize) to false, which will disable user resizing of the window.
