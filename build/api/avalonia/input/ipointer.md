## Definition[​](#definition "Direct link to Definition")

Namespace:[Avalonia.Input](./)

Identifies specific pointer generated by input device.

```csharp
public interface IPointer

```

## Remarks[​](#remarks "Direct link to Remarks")

Some devices, for instance, touchscreen might generate a pointer on each physical contact.

#### Parameters[​](#parameters "Direct link to Parameters")

`control` [Avalonia.Input.IInputElement](xref:Avalonia.Input.IInputElement)

The control.

#### Remarks[​](#remarks-1 "Direct link to Remarks")

When an element captures the pointer, it receives pointer input whether the cursor is within the control's bounds or not. The current pointer capture control is exposed by the [Avalonia.Input.IPointer.Captured](xref:Avalonia.Input.IPointer.Captured) property.
